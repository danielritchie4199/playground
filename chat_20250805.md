  - Claude 3.5 Haiku
  - Claude 3.5 Opus
  - 기타 지원되는 모델들

**🔹 Grok-4 관련:**
- **Grok-4**: xAI에서 개발한 AI 모델

- 현재 Grok-4 전용 확장은 제한적일 수 있음

#### **방법 2: 웹 기반 Grok 사용**
- **xAI 웹사이트**: https://x.ai/ 에서 Grok-4 직접 사용
- **별도 탭**: Cursor와 별도로 Grok-4 사용

#### **방법 3: API 연동**
- Grok-4 API가 공개되면 Cursor에서 연동 가능
- 현재는 제한적 API 접근
